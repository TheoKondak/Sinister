sinisterScss

Introduction

Sinister Scss came out of need for my work workflow. Quite often I had to implement specific layouts, and I had to write the same code again and again. So I thought why not try to write the code once, and save time for ever? And that's how this project begun.

The project evolved over time, by adding new stuff to improve responsive behaviour, typography, responsive typography, as well as a library of animations, dropbox shadows as well as add some consistency in font sizes, spacing etc.

The Configuration

Flex

This is a library containing most of flex box properties in mixins and classes.

How to Use

This is a list of available flex mixins and classes:

  • flex
  • justifyCenter
  • justifyStart
  • justifyEnd
  • justifySpaceBetween
  • justifySpaceEvenly
  • justifySpaceAround
  • wrap
  • nowrap
  • alignStart
  • alignEnd
  • alignBaseline
  • alignCenter
  • alignStretch
  • contentStart
  • contentEnd
  • contentCenter
  • contentBetween
  • contentAround
  • contentStretch
  • flexColumn
  • flexRow
  • flexGrow
  • noGrow
  • flexShrink
  • noShrink
You can use any of the above as a mixin:

Example

Let's say you have the following scss, and you want to make it flex, justify it in the center, and align items in the center.

<div class='containerName'>
<div class='item'>
</div>
<div class='item'>
</div>
</div>
With Classes
<div class='containerName justifyCenter alignCenter'>
<div class='item'>
</div>
<div class='item'>
</div>
</div>
Adding those two classes, will make the element display:flex;
With Mixins
.containerName{
@include justifyCenter;
@include alignCenter;
}
