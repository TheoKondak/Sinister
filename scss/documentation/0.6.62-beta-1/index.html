    [private] contain

    @function contain($data, $val) { ... }

    Description

    This function checks if a list or map contains an arguement. If the arguement is present either as key or value, the function returns true

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $data

    Takes a list or a map.

    		List or Map— none
    $val

    Takes a value of any type and checks if it exists in the current list or map.

    		Map— none

    Returns

    Bool —

    Returns a true if the value is contained in the list or map.

    Example

    Should return false, and thus color will not change to #f31

    $just-a-map: (1: one, 2: two);
  @if contain($just-a-map, three){
   .sin{
    color: #f31;
   }
  }

    spacing

    @mixin spacing($breakpoint, $property, $direction, $value, $scale) { ... }

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    A breakpoint from breakpoints in Sinister configuration.

    		String— none
    $property

    The first letter of a spacing property, m for margin, p for padding, g for gap.

    		String— none
    $direction

    The first letter of a spacing property direction. For margin and padding, t for top, r for right, b for bottom, l for left. Gap property has only generic, column and gap, so if any of t,b is selected row-gap will be applied and similar for column-gap.

    		String— none
    $value

    A spacing value from spacing in Sinister configuration

    		String— none
    $scale

    A number that will be multiplied with the spacing value selected. Tuning this is not suggested. Default value is 1.

    		Number— none

    • Invalid input in spacing mixin. See documentation for more.
